Question-1: What are the different ways to select an element in the DOM?

There are a few ways to select an element in the DOM 1.getElementsByTagName 2. getElementsByClassName 3. getElementById 4.querySelector 5. querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML returns all the HTML content and tags, innerText returns only the text from the text element, textContent returns all the hidden text as well as visible text

Question-3: What is event delegation in the DOM?

This a way to add event listener to a parent element and add a event on its child element.

Question-4: What is event bubbling in the DOM?

Event bubbling is adding an event on child element and it will trigger a bubble to it's parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

To create element we can use document.createElement and to add the newly created element we can append the element to a parent element and to remove an element we can use removeChild.
